# Keyboard Response Time Tester

**Course:** Human Computer Interaction
**Topic:** Human Response Time & Keyboard Interaction

---

## What it Does

A character (number or letter) appears on screen. The user reacts by pressing
the correct key as fast as possible. Response time is recorded in milliseconds.

---

## Hand Placement

| Stimulus | Press | Hand |
|----------|-------|------|
| Number (0–9) | `L` | Right index finger |
| Letter (A–Z) | `A` | Left index finger |

---

## What Gets Recorded

- **Response time** — ms from stimulus shown to key pressed
- **Wrong key press** — flagged as error, time still shown
- **Average time** — tracked across all attempts

---

## How to Use

1. Open `index.html` in browser
2. Place left finger on **A**, right finger on **L**
3. Press **Space** to start
4. React to each stimulus as fast as possible
